# BARKKAI
Customization of https://github.com/BloodHoundAD/BARK

## Addtional functions
1. Get-AllEntraAppsWithOwners

Listing Entra Apps by `Get-AllEntraApps` with owners' userprincipal added.

2. Get-AllEntraRolesWithMembers

With listing Entra Roles by `Get-AllEntraRoles` with their members' userprincipal added.

3. Get-AllEntraServicePrincipalsWithRoles

With listing Entra SPs by `Get-AllEntraServicePrincipals` with their Entra roles added.
