# BARKKAI
Customization of https://github.com/BloodHoundAD/BARK

## Addtional functions
1. Get-AllEntraAppsWithOwners
With listing Apps with `Get-AllEntraApps` owners' userprincipal added.
2. Get-AllEntraRolesWithMembers
With listing Entra Roles with `Get-AllEntraRoles` owners' its members' userprincipal added.
3. Get-AllEntraServicePrincipalsWithRoles
With listing Entra SPs with `Get-AllEntraServicePrincipals` its Entra roles added.
